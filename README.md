# Code Reviewer Plugin for Claude Code

Architecture conformance, code quality, security, and performance review with auto-fix capabilities.

## Installation

```
/plugin marketplace add nagisanzenin/claude-code-plugins
/plugin install code-reviewer@nagisanzenin
```

## Features

- Architecture conformance checking
- Code quality and style analysis
- Security vulnerability detection
- Performance review and optimization
- Auto-fix for common issues
